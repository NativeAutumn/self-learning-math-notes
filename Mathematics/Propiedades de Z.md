Tema: [[Teoría de números]]

$\textbf{El conjunto } \mathbb{Z} \textbf{ con las operaciones } + \textbf{ y } \cdot \textbf{ es un anillo conmutativo con 1.}$ Es decir, exhibe las siguientes propiedades:
$\forall a,b,c \in \mathbb{Z}$
### Aditivas
$1.\; a+b \in \mathbb{Z}$

$2.\; a+(b+c)=(a+b)+c$

$3.\; a+b=b+a$

$4.\; \exists\, 0 \in \mathbb{Z} \text{ tal que } a+0=a$

$5.\; \forall a \in \mathbb{Z},\; \exists\, (-a) \in \mathbb{Z} \text{ tal que } a+(-a)=0$

---

### Multiplicativas
$6.\; ab \in \mathbb{Z}$

$7.\; a(bc)=(ab)c$

$8.\; ab=ba$

$9.\; \exists\, 1 \in \mathbb{Z},\; 1 \neq 0,\; \text{ tal que } a\cdot 1=a$

---

### Distributiva
$10.\; a(b+c)=ab+ac$

$11.\; (a+b)c=ac+bc$

---

### Dominio entero y discreción del orden
$\textbf{Si } a,b \in \mathbb{Z} \textbf{ y } ab=0 \textbf{ entonces } a=0 \textbf{ o } b=0.$

$\nexists\, z \in \mathbb{Z} \text{ tal que } 0<z<1.$
